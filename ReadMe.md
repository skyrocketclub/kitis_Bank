kitis_Bank is a Banking Application built with Node.js. It is capable of providing the users with the APIS with which to Create, Read,Update and Delete accounts stored in a MongoDB Atlas Database.

It also provides APIs that give critical insight with stats like

1. Total money in the bank available for transactions
2. Number of active savings account
3. Number of active Current Account

Concepts applied

1. MongoDB Aggregation Pipeline
2. Alias Routing

Technology Used

1. MongoDB Atlas
2. Express
3. Node.JS
4. Mongoose
